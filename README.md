# Serviço de ordem

Esse serviço é um pedaço do ecosistema de micro serviços de Gerenciamento de Inventário/Produtos.
Sendo esse o serviço de ordem

### Dependências
* Spring (boot)
* Micrometer
* Zipkin
* Eureka Client
* resilience4j
* webflux
* Kafka
* circuitbreaker

É necessário que tenha um server Kafka rodando

### Como rodar
`docker compose up`
